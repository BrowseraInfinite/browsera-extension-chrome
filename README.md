# Browsera-InfiniTab (BIT)

**BrowseraInfiniTab (BIT)** is a minimalist, keyboard-first browser extension that transforms your "New Tab" page into a high-performance command center. As a key project under the **Browsera** ecosystem, BIT is designed for speed, allowing you to navigate the web, launch searches, and manage custom shortcuts without ever touching your mouse.

---

### ✨ Features
* **Lightning-Fast Spotlight:** Invoke the command interface instantly with `Ctrl + Space`.
* **Keyboard-First Workflow:** Navigate the web with precision; search, browse, and manage commands entirely through text input.
* **Command-Driven Navigation:** Built-in search prefixes and custom aliases make jumping between sites instantaneous.
* **Minimalist Aesthetics:** A distraction-free environment that stays out of your way.

---

### 🚀 How to Use

1. **Activate:** Open a new tab (or press `Ctrl + Space` anywhere).
2. **Execute:** Type your command or search query and press `Enter`.
3. **Manage:** Use built-in system commands to expand your workflow.

---

### ⌨️ Command Reference

| Command | Description |
| :--- | :--- |
| `g [query]` | Quick Google Search |
| `y [query]` | Search YouTube |
| `r [query]` | Search Reddit |
| `gh [query]` | Search GitHub |
| `w [url]` | Navigate to a specific website (e.g., `w google.com`) |
| `/add` | Open the menu to create your own custom shortcuts |
| `/list` | View your library of custom and system commands |
| `/delete [key]` | Remove a saved custom shortcut |
| `/help` | Get assistance and community links |

---

### 🛠 Installation Instructions (Chrome)

1. Download the source code from this repository.
2. Navigate to `chrome://extensions/` in your browser.
3. Enable **Developer mode** (toggle in the top-right corner).
4. Click **Load unpacked** and select the folder where you saved the extension files.
5. Open a new tab—BrowseraInfiniTab is now your default interface.

---

### 💡 Pro Tips
* **Custom Shortcuts:** Use `/add` to create shorthand aliases for your favorite websites. When creating a shortcut, use the `w` prefix in the action (e.g., `w discord.com`) to ensure it navigates correctly.
* **Dismissing Views:** Whenever a modal (List/Add/Help) is open, you can instantly return to your new tab by pressing `Esc`.
